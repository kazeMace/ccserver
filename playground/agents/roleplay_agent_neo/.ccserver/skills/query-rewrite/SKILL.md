# Query 改写 Skill

将自然对话中的用户消息改写成清晰、搜索引擎友好的搜索查询词。

**输入**：
- `user_message`：当前用户消息
- `history`（可选）：最近 3-5 轮对话历史（`user/assistant` 交替）
- `date`（**必须**）：当前日期，从上下文中的 `[当前时间]` 字段读取，格式 `YYYY-MM-DD`

> ⚠️ **严禁使用训练知识中的年份**。`{YYYY}`、`{M}` 等时间占位符必须从 `[当前时间]` 字段中提取，不得凭记忆猜测。

---

## 改写流程

### Phase 1 — 指代消解（Coreference Resolution）

扫描用户消息，识别所有代词、省略主语、模糊指示词：

| 类型 | 示例原文 | 来源 | 消解后 |
|------|---------|------|-------|
| 人称代词 | "她最近在干嘛" | 上轮 assistant 提到 "张雨绮" | "张雨绮最近动态" |
| 指示代词 | "那个电影好看吗" | 上轮提到《哪吒之魔童降世》 | "《哪吒之魔童降世》评分 口碑" |
| 省略主语 | "现在多少钱了" | 上轮聊比特币 | "比特币 现在价格" |
| 隐含主题 | "还有新消息吗" | 对话主题：海贼王动画更新 | "海贼王动画 最新更新 2026" |

**规则**：
- 追溯最近 5 轮历史，优先最近 1-2 轮
- assistant 消息中的「主动提及」与 user 消息中的「明确讨论」权重相同
- 消解失败（歧义无法确定）→ 保留原词，不猜测

---

### Phase 2 — 意图识别

识别用户真实意图，分三类：

| 意图类型 | 判断标准 | 时间处理 |
|---------|---------|---------|
| **回顾型** | 问"已发生/已发布"的事："最近有啥新电影"、"昨天发生了什么" | 只保留 ≤ 今天 |
| **展望型** | 问"即将/计划"的事："有什么演唱会"、"这周末能干嘛" | 只保留 ≥ 今天 |
| **混合型** | 横跨过去与未来："最近有什么好玩的" | 保留前后各 2 周 |
| **事实型** | 问固定知识："XX 是什么意思"、"谁是 XX" | 不加时间词 |
| **天气型** | 询问天气：任何"热不热"、"今天天气"、"下雨吗"及问候语；**第一轮对话（无历史）强制归为此类** | 直接用城市名，不走搜索 |

---

### Phase 3 — Query 构造规则

**实时信息类**（回顾/展望/混合）：

```
{消解后的核心实体} + {时间词} [+ {意图修饰词}]
```

时间词模板（根据意图类型选择，**YYYY/M 均取自 `[当前时间]`**）：
- 回顾型：`{YYYY}年{M-1}-{M}月` 或 `{YYYY}年{M}月`
- 展望型：`{YYYY}年{M}月 即将` 或 `{YYYY}年{M}月 预告`
- 混合型：`{YYYY}年{M}月`

**生词/人名/机构类**：
```
{词} 是什么     # 生词
{人名} 最新动态 {YYYY}   # 真实人物
{机构} 简介             # 机构
```

**上下文承接补充**：
- 如果本轮消息是对上轮内容的**追问/深入**，在 query 末尾追加核心上下文词
- 示例：上轮聊了某歌手发新歌，用户说"歌词什么意思"→ `{歌手名} {歌名} 歌词 含义`

---

### Phase 4 — 搜索引擎友好化

最终 query 必须满足：

1. **无口语词**：去掉"啊"、"呀"、"吧"、"诶"、"哇"等语气词
2. **无疑问句结构**：不用"什么"、"怎么"开头的完整问句，改为关键词组合
3. **长度控制**：10-25 字，超过则保留核心实体 + 时间词 + 1 个限定词
4. **实体优先**：人名、作品名、地名保持完整，不可缩写
5. **歧义消除**：加领域词区分同名实体，如"苹果 手机" vs "苹果 水果"

---

## 输出格式

```
[Query 改写]
原始消息：{user_message}
指代消解：{消解说明，无则写"无需消解"}
用户意图：{意图类型} — {一句话描述用户真实想查什么}
改写 Query：{最终 query}
搜索工具：{search_web / search_news / get_weather / 无需搜索}
时间过滤：{回顾型 / 展望型 / 混合型 / 不适用}
```

## 示例

**示例 1 — 指代消解**

```
历史：
  assistant: 最近周杰伦发了新专辑《最伟大的作品》
  user: 里面有啥好听的歌？

原始消息：里面有啥好听的歌？
指代消解："里面" → 周杰伦《最伟大的作品》专辑
用户意图：回顾型 — 想了解该专辑的曲目/推荐
改写 Query：周杰伦 最伟大的作品 专辑 歌曲推荐
搜索工具：search_web
时间过滤：不适用（事实型）
```

**示例 2 — 展望型 + 上下文承接**

```
历史：
  user: 你喜欢看演唱会吗
  assistant: 超喜欢的！上次看了五月天

原始消息：最近有什么演唱会

指代消解：无需消解
用户意图：展望型 — 查询近期即将举办的演唱会
改写 Query：演唱会 2026年3月 即将
搜索工具：search_news
时间过滤：展望型（≥ 今天）
```

**示例 3 — 天气 + 问候触发**

```
历史：（无）
原始消息：最近天气怎么样

指代消解：无
用户意图：天气型 — 查询当前天气（城市未知，用用户画像/角色所在城市）
改写 Query：{城市}（从用户画像或角色人设获取，默认上海）
搜索工具：get_weather
时间过滤：不适用
```
